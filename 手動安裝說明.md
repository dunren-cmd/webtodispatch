# Supabase CLI 手動安裝說明

由於 PowerShell 環境可能有問題，以下是手動安裝 Supabase CLI 的步驟：

## 方法 1：使用安裝腳本（推薦）

1. **以系統管理員身分開啟 PowerShell**
   - 按 `Win + X`
   - 選擇「Windows PowerShell (系統管理員)」或「終端機 (系統管理員)」

2. **執行安裝腳本**
   ```powershell
   cd "c:\Users\dunre\OneDrive\文件\請購單\WebToDispatch_2"
   .\安裝_Supabase_CLI.ps1
   ```

3. **如果出現執行原則錯誤**
   ```powershell
   Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
   ```
   然後再次執行安裝腳本

## 方法 2：完全手動安裝

### 步驟 1：下載 Supabase CLI

1. 開啟瀏覽器，前往：https://github.com/supabase/cli/releases
2. 找到最新版本（例如：v1.xxx.x）
3. 下載 `supabase_windows_amd64.zip`

### 步驟 2：解壓縮

1. 解壓縮下載的 ZIP 檔案
2. 會得到 `supabase.exe` 檔案

### 步驟 3：放置執行檔

**選項 A：放到系統資料夾（需要管理員權限）**

1. 以系統管理員身分開啟檔案總管
2. 將 `supabase.exe` 複製到：`C:\Windows\System32`

**選項 B：建立專用資料夾並加入 PATH**

1. 建立資料夾：`C:\Tools\supabase`
2. 將 `supabase.exe` 放到這個資料夾
3. 將資料夾加入系統 PATH：
   - 按 `Win + R`，輸入 `sysdm.cpl`，按 Enter
   - 點擊「進階」標籤
   - 點擊「環境變數」
   - 在「系統變數」中找到 `Path`，點擊「編輯」
   - 點擊「新增」，輸入 `C:\Tools\supabase`
   - 點擊「確定」儲存所有變更

### 步驟 4：驗證安裝

1. **重新開啟 PowerShell**（讓 PATH 變更生效）
2. 執行：
   ```powershell
   supabase --version
   ```
3. 應該會顯示版本號，例如：`supabase version 1.xxx.x`

## 方法 3：使用 Chocolatey（如果已安裝）

```powershell
choco install supabase
```

## 方法 4：使用 winget（Windows 10/11）

開啟命令提示字元（cmd）或 PowerShell，執行：

```cmd
winget install --id=Supabase.CLI
```

## 驗證安裝

安裝完成後，在任何新的 PowerShell 視窗中執行：

```powershell
supabase --version
```

如果顯示版本號，表示安裝成功！

## 下一步

安裝完成後，可以繼續進行：

1. 初始化 Supabase 專案：
   ```powershell
   cd "c:\Users\dunre\OneDrive\文件\請購單\WebToDispatch_2"
   supabase init
   ```

2. 建立資料庫結構：
   ```powershell
   supabase migration new init
   ```

3. 啟動本地 Supabase：
   ```powershell
   supabase start
   ```

## 疑難排解

### 問題：執行 `supabase` 時顯示「找不到命令」

**解決方案：**
1. 確認 `supabase.exe` 在 PATH 中
2. 重新開啟 PowerShell（讓 PATH 變更生效）
3. 檢查 PATH：
   ```powershell
   $env:Path -split ';' | Select-String supabase
   ```

### 問題：權限不足

**解決方案：**
- 以系統管理員身分執行 PowerShell
- 或將 supabase.exe 放到不需要管理員權限的位置（如用戶資料夾），然後將該位置加入用戶 PATH

### 問題：下載速度慢

**解決方案：**
- 使用瀏覽器直接下載，然後手動解壓縮
- 或使用下載工具（如 IDM、迅雷等）
