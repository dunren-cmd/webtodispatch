# WebToDispatch_2 推送到 Git 說明

## 快速推送步驟

### 方法 1：使用批次檔（最簡單）

1. 在檔案總管中，進入 `WebToDispatch_2` 資料夾
2. 雙擊執行 `push_to_git.bat`
3. 等待執行完成

### 方法 2：使用命令提示字元（CMD）

1. 開啟命令提示字元（CMD）
2. 執行以下命令：

```cmd
cd /d "c:\Users\dunre\OneDrive\文件\請購單\WebToDispatch_2"
git add .
git commit -m "更新：WebToDispatch_2 專案"
git push origin main
```

### 方法 3：使用 PowerShell

1. 開啟 PowerShell
2. 執行以下命令：

```powershell
cd "c:\Users\dunre\OneDrive\文件\請購單\WebToDispatch_2"
```

## 如果遇到問題

### 問題 1：認證失敗

如果推送時要求輸入帳號密碼：
- 使用 GitHub Personal Access Token 作為密碼
- 或設定 SSH 金鑰

### 問題 2：沒有變更需要提交

如果顯示 "nothing to commit"，表示所有檔案都已經提交過了。

### 問題 3：遠端分支不存在

如果顯示分支錯誤，可以執行：
```cmd
git push -u origin main
```

## 檢查狀態

在推送前，可以先檢查狀態：
```cmd
git status
```
